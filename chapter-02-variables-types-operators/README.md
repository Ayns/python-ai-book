# Chapter 2: Variables, Data Types & Operators
Strings, integers, floats, booleans. Type conversion, comparison operators, and f-string formatting.

## Project: Calculator
Run: `python calculator.py`

## What You'll Learn
- 4 data types: `str`, `int`, `float`, `bool`
- Type conversion: `int()`, `float()`, `str()`
- Comparison: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Logical operators: `and`, `or`, `not`
- f-string formatting: `{value:,.2f}`
