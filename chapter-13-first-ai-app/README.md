# Chapter 13: Your First AI App
OpenAI and Anthropic API calls, streaming, multi-turn conversations, vision/multimodal, API key security.

## Project: AI Chatbot
Run: `python chatbot.py`

## Requirements
```
pip install openai python-dotenv
```

## Setup
Create a `.env` file:
```
OPENAI_API_KEY=your-key-here
```

## What You'll Learn
- Making API calls to OpenAI
- Multi-turn conversation with history
- Streaming responses
- Sending images to AI (vision)
- API key security
