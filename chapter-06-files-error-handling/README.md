# Chapter 6: File Handling & Error Management
Reading/writing files, CSV, JSON, and try/except error handling. Your programs gain permanent memory.

## Project: Expense Tracker
Run: `python expense_tracker.py`

## What You'll Learn
- `open()` with `"r"`, `"w"`, `"a"` modes
- CSV files with the `csv` module
- JSON files with `json.dump()` and `json.load()`
- `try`/`except` for error handling
- Data persists between sessions!
